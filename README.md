## 🌊 What is SMZ Player?

Most music apps make you think. SMZ Player makes you *feel*.

Instead of typing song names into a rigid search bar, just tell it your vibe — **"Coding 💻"**, **"Anime 🌸"**, **"Phonk 🏎️"** — and SMZ Player instantly pulls up the perfect soundtrack for that moment. No playlists to scroll. No mood mismatch. Just the right music, right now.

> *"Built for people who live in the moment — not in the music library."*

## 🚀 Getting Started

Clone it. Run it. Vibe.

```bash
# 1. Clone the repo
git clone https://github.com/shaanahmed/smz-musicapp.git
cd smz-musicapp

# 2. Install dependencies
npm install   # or pip install -r requirements.txt

# 3. Set your environment variables
cp .env.example .env
# → Add your JioSaavn API instance URL

# 4. Launch
npm start
```

> **Note:** You'll need a deployed instance of the [JioSaavn API](https://github.com/sumitkolhe/jiosaavn-api) to power the music streaming. Point your `.env` to it and you're good to go.
