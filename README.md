SMZ Player
A sleek, modern music streaming web application designed to bridge the gap between your current mood and the perfect soundtrack. SMZ Player moves away from traditional, rigid search structures to provide a fluid, intuitive, and "vibe-centric" listening experience.

💡 The Vision
The inspiration behind SMZ Player was to create a high-level music tool that understands context. Instead of manually scrolling through endless playlists, the app allows users to input how they are feeling or what they are doing—whether it's "Coding 💻", "Anime 🌸", or "Phonk 🏎️"—and instantly delivers a curated selection of tracks that match that specific energy.

This project represents a core milestone in my journey toward becoming a Global Architect in AI and Web3, focusing on seamless user experiences and robust API integration.

✨ Key Features
Vibe-Based Discovery: Utilize dedicated "AI" and "Vibe" buttons to find music that matches your emotional state or current activity.

Intelligent Search: A powerful search interface that handles everything from specific artists (like Eminem) to broad mood descriptions.

Personalized Library:

Liked ❤️: Save your favorite tracks for instant access.

Queue ✨: Manage and view your upcoming tracks on the fly.

Neo-Brutalist UI: A high-contrast, modern design that ensures the player is both visually striking and easy to navigate on any device.

Smart Playback Controls: Includes high-fidelity playback, shuffle, repeat, and a dynamic progress bar for precise navigation.

Track Insights: Deep-dive into song metadata with the integrated "Info" toggle.

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript (Neo-brutalist/Glassmorphism design).

Backend: Python-based logic for handling data and streaming.

API: Deployed instance of the JioSaavn API.

Deployment: Vercel (for high availability and continuous delivery).
